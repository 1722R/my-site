Python 基础语法

了解 Python 的基本语法规则，包括变量、数据类型、运算符和控制结构

什么是 Python？

Python 是一种简洁、易读的高级编程语言，由 Guido van Rossum 于 1991 年发布。它以代码可读性强、语法简洁著称，广泛应用于 Web 开发、数据分析、人工智能、自动化脚本等领域。

💡 为什么选择 Python？

Python 语法接近自然语言，学习曲线平缓，是编程入门的最佳选择之一。同时它拥有丰富的第三方库生态，可以应对几乎任何编程需求。

Python 中不需要声明变量类型，直接赋值即可。变量名由字母、数字和下划线组成，不能以数字开头。

# 输出变量

数据结构

掌握 Python 中最常用的数据结构：列表、元组、字典和集合

列表（List）

列表是 Python 中最常用的数据结构，用方括号 [] 表示，可以存储任意类型的元素。

Python 
fruits = ["苹果", "香蕉", "橙子"]

# 增加元素
fruits.append("葡萄")          # 末尾添加
fruits.insert(1, "西瓜")        # 索引1处插入
fruits.extend(["芒果", "草莓"])  # 合并列表

# 删除元素
fruits.remove("香蕉")   # 按值删除
popped = fruits.pop()    # 删除末尾元素

# 列表推导式
squares = [x**2 for x in range(10)]
evens = [x for x in range(20) if x % 2 == 0]

print(len(fruits))     # 列表长度
print(sorted(fruits))  # 排序
print("苹果" in fruits)  # True

元组（Tuple）

元组与列表类似，但一旦创建就不可修改。用圆括号 () 表示。

Python 
point = (3, 4)
rgb = (255, 128, 0)
single = (42,)   # 单元素元组必须加逗号

# 元组解包
x, y = point
print(x, y)  # 3 4
💡 列表 vs 元组

列表可变，适合需要频繁修改的数据；元组不可变，速度更快，适合存储常量数据，且可以作为字典的键。

字典（Dictionary）

字典是键值对的无序集合，用花括号 {} 表示，通过键快速访问值。

Python 
student = {
    "name": "张三",
    "age": 20,
    "major": "计算机科学",
    "scores": {"math": 90, "english": 85}
}

print(student["name"])           # '张三'
print(student.get("gpa", 0))    # 0（默认值）
student["age"] = 21               # 修改值
student["email"] = "zhangsan@email.com"  # 添加

# 遍历字典
for key, value in student.items():
    print(f"{key}: {value}")

# 字典推导式
squares_dict = {x: x**2 for x in range(6)}

集合（Set）

集合是无序、不重复元素的容器，常用于去重和数学运算。

Python 
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

print(a & b)        # {4, 5}       交集
print(a | b)        # {1,2,3,4,5,6,7,8}  并集
print(a - b)        # {1, 2, 3}    差集
print(a ^ b)        # {1,2,3,6,7,8}  对称差集

# 去重
nums = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(nums))  # [1, 2, 3, 4]

定义函数

函数是可重复使用的代码块，使用 def 关键字定义。

Python 
def greet(name):

def power(base, exp=2):
    return base ** exp

print(power(2, 10))  # 1024

可变参数

Python 
# *args - 接收任意数量的位置参数
def calculate_sum(*args):
    return sum(args)

print(calculate_sum(1, 2, 3))        # 6
print(calculate_sum(10, 20, 30, 40))  # 100

# **kwargs - 接收任意数量的关键字参数
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="张三", age=20, major="CS")

Lambda 表达式

面向对象编程

理解类与对象、继承、封装和多态等核心概念

类与对象

Python 
class Student:
    school = "广东科学技术职业学院"  # 类变量

    def __init__(self, name, age, major):
        self.name = name
        self.age = age
        self.major = major
        self.scores = []

    def add_score(self, score):
        self.scores.append(score)

    def average(self):
        if not self.scores: return 0
        return sum(self.scores) / len(self.scores)

    def __str__(self):
        return f"{self.name（{self.major），平均分：{self.average():.1f}"

s1 = Student("张三", 20, "计算机科学")
s1.add_score(90)
s1.add_score(85)
s1.add_score(92)
print(s1)  # 张三（计算机科学），平均分：89.0

继承

Python 
class Animal:
    def __init__(self, name, sound):
        self.name = name
        self.sound = sound

    def speak(self):
        return f"{self.name说："{self.sound}"

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name, "汪汪！")
        self.breed = breed

class Cat(Animal):
    def __init__(self, name, color):
        super().__init__(name, "喵~")
        self.color = color

# 多态
for animal in [Dog("旺财", "金毛"), Cat("咪咪", "橘色")]:
    print(animal.speak())

封装

Python 
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance  # 私有属性

    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount

    def withdraw(self, amount):
        if amount > self.__balance:
            print("余额不足！")
        else:
            self.__balance -= amount

    @property
    def balance(self):
        return self.__balance

account = BankAccount("张三", 1000)
account.deposit(500)
account.withdraw(200)
print(account.balance)  # 1300

文件操作与异常处理

学习如何读写文件以及使用异常处理机制编写健壮的程序

文件读写

Python 
# 写入文件
with open("hello.txt", "w", encoding="utf-8") as f:
    f.write("Hello, Python!\n")

# 读取文件
with open("hello.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# 逐行读取
with open("hello.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

# 追加内容
with open("hello.txt", "a", encoding="utf-8") as f:
    f.write("追加的新内容\n")
💡 文件打开模式

"r" 只读，"w" 写入（覆盖），"a" 追加，"b" 二进制模式，"+" 读写模式。始终建议指定 encoding="utf-8"。

异常处理

Python 
try:
    num = int(input("请输入一个数字："))
    result = 100 / num
    print(f"结果是：{result}")
except ValueError:
    print("错误：请输入有效的数字！")
except ZeroDivisionError:
    print("错误：不能除以零！")
except Exception as e:
    print(f"未知错误：{e}")
else:
    print("计算成功！")
finally:
    print("程序结束。")

综合实战

Python 
# 自定义异常
class ScoreError(Exception):
    """分数异常"""
    pass

def validate_score(score):
    if not isinstance(score, (int, float)):
        raise TypeError("分数必须是数字")
    if score < 0 or score > 100:
        raise ScoreError(f"分数 {score} 不在 0-100 范围内")
    return True

# 读取成绩文件并统计
def process_scores(filepath):
    scores = []
    try:
        with open(filepath, "r", encoding="utf-8") as f:
            for line in f:
                score = int(line.strip())
                validate_score(score)
                scores.append(score)
    except FileNotFoundError:
        print(f"文件 {filepath} 不存在")
    except (ValueError, ScoreError) as e:
        print(f"数据格式错误：{e}")
    else:
        print(f"平均分：{sum(scores)/len(scores):.1f}")
        print(f"最高分：{max(scores)}")
        print(f"最低分：{min(scores)}")
